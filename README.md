exercícios de Lógica de Programação em C
Este repositório contém uma coleção de desafios e exercícios práticos desenvolvidos durante o meu primeiro semestre de contato com a linguagem C. O objetivo principal é consolidar fundamentos de algoritmos e aprofundar o entendimento sobre manipulação de dados e fluxo de controle.

Conceitos Aplicados:
Durante o desenvolvimento destes exercícios, explorei os pilares da linguagem:

Entrada e Saída: Uso de scanf e printf.
Estruturas Condicionais: Tomada de decisão com if/else e switch.
Laços de Repetição: Automação com for, while e do-while.
Modularização: Criação de funções para reutilização de código.
Gerenciamento de Memória: Introdução a ponteiros e vetores (arrays).

Estrutura do Repositório
O projeto está organizado por nível de complexidade ou temática:
/Basicos: Operações aritméticas e tipos de dados simples.
/Condicionais: Desafios de lógica booleana.
/Repeticao: Exercícios envolvendo somatórios, contadores e validações.
/Matrizes_e_Vetores: Manipulação de conjuntos de dados.
